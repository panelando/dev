# dev
Setup for development

## Installing Git

1. Download Git Bash for windows: https://git-for-windows.github.io/
2. Open Git Bash.
3. Setup github access: https://help.github.com/articles/generating-an-ssh-key/

## Installing Node.js and npm

1. Download and install node: http://blog.teamtreehouse.com/install-node-js-npm-windows
2. Install NVM (Node Version Manager): https://github.com/creationix/nvm#manual-install
3. Install node versions using nvm:
  `nvm install 0.12.1`, `nvm install stable`
